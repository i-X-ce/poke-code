ルービックキューブで遊ぶことができるプログラムです。  
こちらの動画で紹介しています。

## 操作方法
表示されたルービックキューブの展開図は、以下の操作で楽しむことができます。

| 操作対象           | ボタン   | 機能                                             |
| ----------------- | ------- | ----------------------------------------------- |
| 回転記号にカーソル | Aボタン  | カーソルのある箇所のルービックキューブが回転します。 |
| 回転記号にカーソル | Bボタン  | Aボタンとは逆方向にルービックキューブが回転します。 |
| メニュー呼び出し   | STARTボタン | 「スクランブル」または「やめる」を選択できます。   |

## スクランブル機能
STARTボタンメニューで「スクランブル」を選択すると、ルービックキューブが自動で20回混ぜられます。スクランブル後には、実行された回転符号が画面に表示され、実際のルービックキューブでもその手順を試すことが可能です。
混ぜられたキューブをプレイヤーが揃えると、「クリアー！」のメッセージが表示されます。

## 注意点
本プログラムのスクランブル機能は、一般的なルービックキューブのスクランブルとは異なり、必ずしも最適な手順ではない混ぜ方をする場合があります。あらかじめご了承ください。
