## 操作方法

| 操作   | 内容            |
| ------ | --------------- |
| ↑      | アドレス -07h   |
| ↓      | アドレス +07h   |
| ←      | アドレス -01h   |
| →      | アドレス +01h   |
| B+↑    | アドレス -100h  |
| B+↓    | アドレス +100h  |
| B+←    | アドレス -1000h |
| B+→    | アドレス +1000h |
| A+↑    | 中身 -01h       |
| A+↓    | 中身 +01h       |
| A+←    | 中身 -10h       |
| A+→    | 中身 +10h       |
| START  | 終了            |
| SELECT | コードの実行    |

## 注意事項

本バイナリエディタの開始位置はDE64hです。
この開始位置周辺を書き換えるプログラムと併用すると、本バイナリエディタが正常に動作しなくなる可能性がありますのでご注意ください。
