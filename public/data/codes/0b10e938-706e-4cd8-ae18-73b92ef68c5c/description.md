ゲーム内で再生されている音楽に合わせて視覚的なエフェクトを表示するデモンストレーションです。  
この動画で紹介しています。

## 機能説明
実行すると、現在鳴っている音楽の音程や長さに応じて、画面上に四角い枠が次々と描画されます。

*   **音程と表示:** 音が高いほど、枠はより細く、画面の上方に表示されます。
*   **音の長さと表示:** 音が長いほど、描画される枠も長く表示されます。

## 操作方法
| 操作内容     | 説明                       |
| :----------- | :------------------------- |
| 任意のボタン | プログラムを中断し、元の状態に戻ります。 |

## 注意事項
*   一部の状況で音が鳴っていても枠が表示されない場合がありますが、これは現在の仕様です。
*   プログラム終了後、画面表示が一時的に乱れることがございます。キャラクターを移動させることで元の状態に戻りますのでご留意ください。
