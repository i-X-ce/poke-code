GeoGuesser風のゲームを起動するコードです。こちらの動画で紹介しています。

## 操作方法

| 手順 | 操作内容                                                                              | 備考                                                                                                            |
| ---- | ------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| 1    | プログラムを起動します。                                                              | ゲームが開始されます。                                                                                          |
| 2    | ランダムなマップのランダムな座標に移動します。                                        | 特記事項なし                                                                                                    |
| 3    | Aボタンを押します。                                                                   | マップが表示されます。Bボタンを押すと、ゲームを中断する選択肢が表示されます。                                   |
| 4    | マップ上で正しいと思われる位置にカーソルを移動させ、Aボタンを押して位置を決定します。 | もう一度マップを表示したい場合はBボタンを押します。この際、プレイヤーの現在位置がランダムな座標に再移動します。 |
| 5    | 正解/不正解の結果を確認します。                                                       | 正解の場合、次のマップへ進みます。不正解の場合、正解数に応じた「なかよしバッヂ」を獲得します。                  |
